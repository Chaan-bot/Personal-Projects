## Project Name: DJAPP

### Description:
Sound mixing and DJ App . 

### Setup Instructions:

1. **Install JUCE:**
   - Download and install JUCE from [juce.com](https://juce.com/get-juce).
   - Extract it to a local directory (e.g., `C:/JUCE`).

2. **Configuration:**
   - Ensure JUCE is added to your include path. Example for VSCode:
     ```
     C:/JUCE/modules
     ```

3. **Run the Project:**
   - Build the project in your IDE after configuring the JUCE path.

### Notes:
- JUCE is not included in this repository, install it locally.
